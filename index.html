Lab 1

  1. In your Amazon Workspace, click the ‘Search’ icon and type “PowerShell”.
  2. Choose the top choice “Windows PowerShell” and make it full screen.
  3. Let’s run some commands. Note that we can only type and enter a command one line at a time.
     a. In the “Windows PowerShell Window”, type Get-Help and press the Enter/Return key.
     Hint: Get-Help is a PowerShell command that displays help information about PowerShell commands and concepts. PowerShell Help describes PowerShell cmdlets, functions, scripts, and modules, and explains concepts, including the elements of the PowerShell language

    4.  Run:

     b. In the “Windows PowerShell Window”, type Get-Alias and press the Enter key.
     Hint: The Get-Alias command provides a list of pre-defined aliases for commands that you can enter in place of the full command names (ex. You can type TNC instead of writing out the Test-NetConnection command, allowing you to save time in the console)

    Run:

  4. Close ‘Windows PowerShell’.


  5. Now let’s compare that to Windows PowerShell ISE!
    In your Amazon Workspace, click the “Search” icon and type “PowerShell”. Choose “Windows PowerShell ISE”.
  6. In the “Windows PowerShell ISE”, we’re going to turn on the script pane.

     a. Click 'View'
     b. Click 'Show Script Pane'
  7. Let’s run some commands. Hint: Note how it displays and interacts differently than Windows PowerShell Console!
      a. In the white box at the top, type Get-Help. Click on the green ‘Play’ button in the task bar to run the command.
       Note: it may take up to 30 seconds for the play button to become available, just be patient!

    Run:

     b. In the white box at the top, delete the Get-Help command and type Get-Alias. Click on the green ‘Play’ button in the task bar to run the command.

    Run:
  8. With Windows PowerShell ISE, we can run multiple lines of commands. to do so,
      a. In the white box at the top, type Get-Help and press the Enter/Return key. Then type, Get-Alias
      b. Click on the green 'Play' button in the task bar to run the command.
      c. Scroll up in the bottom blue area and note that both commands ran successfully!

    Run:
  9. Close Windows PowerShell ISE.

    Congratulations! You completed Lab 1!
